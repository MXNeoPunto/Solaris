# Solaris
## Google Play Billing Setup
To enable the "Remove Ads" feature for $1.99 in the Google Play Store:
1. Go to your Google Play Console and select the Solaris app.
2. Navigate to **Monetize > Products > In-app products**.
3. Create a new product.
4. Set the Product ID to exactly: `remove_ads_199`
5. Set the price to $1.99.
6. Activate the product.
